# ChemQuest Organic v2

有機化学の基本事項暗記・一問一答・共通テスト形式・合成マップ学習アプリです。

## 追加された機能

- ホーム画面
- 一問一答モード
- 共通テストチェックモード
- 合成マップ
- 図鑑
- 苦手復習
- カテゴリ別習熟度
- 連続正解数
- 進捗リセット
