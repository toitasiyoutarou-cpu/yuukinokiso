# ChemQuest Organic Self Study v2 Expanded

有機化学の自学クイズアプリです。

## 改良点
- 単元カードに問題数を表示
- 穴埋め形式は選択問題として実装
- 一問一答・正誤・選択・製法クイズを単元ごとに収録
- 合成マップ・図鑑・学習記録つき

## 収録数
- 一問一答: 90
- 正誤問題: 48
- 選択問題: 145
- 製法・反応ルート: 40
- 合計: 323

## v3-cardstart

単元カードから直接、選択問題を開始できます。カード内の小ボタンから一問一答・正誤・選択・製法にも直接入れます。

## v4-random-prev

- 選択問題の選択肢を表示ごとにランダム化
- 一問一答・正誤・選択・製法クイズに「前の問題」ボタンを追加
