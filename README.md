# ChemQuest Organic Self Study v2 Expanded

有機化学の自学クイズアプリです。

## 改良点
- 単元カードに問題数を表示
- 穴埋め形式は選択問題として実装
- 一問一答・正誤・選択・製法クイズを単元ごとに収録
- 合成マップ・図鑑・学習記録つき

## 収録数
- 一問一答: 90
- 正誤問題: 48
- 選択問題: 145
- 製法・反応ルート: 40
- 合計: 323

## v3-cardstart

単元カードから直接、選択問題を開始できます。カード内の小ボタンから一問一答・正誤・選択・製法にも直接入れます。

## v4-random-prev

- 選択問題の選択肢を表示ごとにランダム化
- 一問一答・正誤・選択・製法クイズに「前の問題」ボタンを追加

## v5-plausible-choices

- 選択問題の誤答選択肢を問題タイプ別に整理
- 官能基・分子式・確認反応・製法など、同じ種類の選択肢が並ぶように調整
- 製法クイズは同じ単元内の生成物を優先して選択肢化


## v6-functional-groups
- 官能基を大幅追加
- 名称→構造、構造→名称、分類、確認反応、構造推定を増強
- 製法・合成ルートを追加
- 既存問題は減らさず追加
